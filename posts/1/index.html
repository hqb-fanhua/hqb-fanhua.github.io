搭建过程

注册GitHub账号

​ 注册GitHub账号可以使用GitHub的远程仓库来部署项目，部署成功后可以进行网络的访问

​ GitHub的官网链接链接

安装Git管理工具

​ 我们将项目部署到github的时候需要git管理工具，使用git可以将创建好的博客文章推送到GitHub远端服务器。

​ Git管理工具需要下载安装下载地址,下载64位系统，直接一路点击下一步快速安装。

下载Hugo项目到本地

​ 在GitHub的仓库中下载Hugo项目到本地电脑，直接下载64位，下载链接

​

​ 下载后进行解压，可以解压到任意的盘上，会得到两个文件和一个运行程序。

​

​ 需要将路径添加到path环境变量中，方便操作。添加步骤为右击我的电脑—>属性—->高级系统设置—>环境变量—>path。我的解压为D:\hugo\bin,则将此路径添加到path。需要注意的是添加路径的格式为 ;D:\hugo\bin

  1. 创建Hugo的站点

    使用Win+R,输入cmd打开命令行，使用hugo的命令查看是否成功安装

    hugo version

    安装成功会显示相应的版本。

    在指定文件下创建站点文件

    hugo new site myblog

  2. 下载Hugo主题

    在主题网站下载一个主题。主题网站

    切换到相应的博客站点路径，例如我刚才创建的是myblog，需要使用cd命令切换到站点内。

    cd [创建的站点目录]

  3. 启动博客本地预览

    使用命令启动博客进行预览

    hugo server -t m10c --buildDrafts

    浏览器输入localhost:8080打开博客

  4. 创建Hugo博客文章

    使用命令创建文章

    hugo new post/blog.md

    在content文件夹下使用命令

  5. 部署到远程仓库

    需要使用github仓库来搭建远程仓库

    使用命令关联远程仓库

    hugo --theme=m10c --baseUrl="https://hqb-fanhua.github.io" --buildDrafts

    成功后目录下会生成public文件夹

  6. 发布博客

​ 在public文件夹下使用命令将文章推送到远程仓库。

git init
git add .
git commit -m "第一次博客内容提交"
git remote add origin http://github.com/hqb-fanhua/hqb-fanhua.github.io.git
git push -u origin master
