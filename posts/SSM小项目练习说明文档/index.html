SSM小项目练习

功能介绍

1.商品查询
2.商品添加
3.订单查询
4.订单分页查询
5.订单详情查询
6.用户管理
7.角色管理
8.资源权限管理
9.权限关联控制
10.AOP日志处理

数据库表设计

1.产品表
序号 字段名称 字段类型 字段描述
1 id varchar(32) 主键id
2 productNum varchar(50) 产品编号
3 productName varchar(50) 产品名称
4 cityName varchar(250) 出发城市
5 DepartureTime timestamp 出发时间
6 productPrice number 产品价格
7 productDesc varchar(500) 产品描述
8 productStatus int 状态(0 关闭 1 开启)
2.订单表
序号 字段名称 字段类型 字段描述
1 id varchar(32) 主键id
2 orderNum varchar(50) 订单编号
3 orderTime timestamp 下单时间
4 peopleCount int 出行人数
5 orderDesc varchar(500) 订单描述
6 payType int 支付方式(0 支付宝 1 微信 2 其他)
7 orderStatus int 订单状态(0 未支付 1 已支付)
8 productid int 产品id(外键)
9 memberid int 会员id(外键)
3.会员表
序号 字段名称 字段类型 字段描述
1 id varchar(32) 主键id
2 name varchar(20) 姓名
3 nickName varchar(20) 昵称
4 phoneNum varchar(20) 电话号码
5 email varchar(50) 邮箱
4.旅客表
序号 字段名称 字段类型 字段描述
1 id varchar(32) 主键id
2 name varchar(20) 姓名
3 sex varchar(20) 性别
4 phoneNum varchar(20) 电话号码
5 credentialsType int 证件类型 0 身份证 1护照 2 军官证
6 credentialsNum varchar(50) 证件号码
7 travellerType int 旅客类型(0 成人 1 儿童)
5.用户表
序号 字段名称 字段类型 字段描述
1 id varchar(32) 主键id
2 email varchar(32) 邮箱
3 username varchar(32) 用户名
4 password varchar(32) 密码
5 phoneNum varchar(32) 电话号码
6 status int 状态(0 未开启 1 开启)
6.角色表
序号 字段名称 字段类型 字段描述
1 id varchar(32) 主键id
2 roleName varchar(32) 角色名
3 roleDesc varchar(32) 角色描述
7.资源权限表
序号 字段名称 字段类型 字段描述
1 id varchar(32) 主键id
2 permissionName timestamp 权限名
3 url varchar(32) 资源路径
8.日志表
序号 字段名称 字段类型 字段描述
1 id varchar(32) 主键
2 visitTime timestamp 访问时间
3 username varchar(32) 操作者用户名
4 ip varchar(32) 访问ip
5 url varchar(32) 访问资源地址
6 executionTime int 执行时长
7 method varchar(32) 执行方法
